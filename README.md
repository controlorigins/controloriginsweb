# ControlOrigins.com Repository

This repository contains the source code for the ControlOrigins.com website.
The website is built using PUG templates and managed with NPM.

## Installation

To install and run the ControlOrigins.com website locally, follow these steps:

Clone the repository to your local machine using the following command:

```cmd
    git clone <repository_url>
```

Navigate to the cloned directory:

```cmd
    cd ControlOrigins.com
```

Install the project dependencies:

```cmd
    npm install
```

## Development

To run the website locally for development purposes, use the following command:

```cmd
    npm run dev
```

This will start a local development server and automatically open the website in your default browser. Any changes you make to the source files will be automatically reflected in the browser.

## Building for Production

To build the ControlOrigins.com website for production, execute the following command:

```cmd
    npm run build
```

This will generate optimized and minified assets in the dist directory, ready for deployment.
